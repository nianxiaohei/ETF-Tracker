# Claude Code 快速启动指南

## ✅ 插件状态
- **插件名称**: Claude Code for VS Code
- **版本**: 2.1.27
- **状态**: ✅ 已安装
- **安装位置**: `~/.cursor/extensions/anthropic.claude-code-2.1.27-darwin-arm64`

## 🚀 启动方法

### 方法 1: 快捷键（最快）
按 `Cmd + Escape` 键即可打开 Claude Code

### 方法 2: 命令面板
1. 按 `Cmd + Shift + P` 打开命令面板
2. 输入 "Claude Code"
3. 选择以下任一选项：
   - `Claude Code: Open in Side Bar` - 在侧边栏打开
   - `Claude Code: Open in New Tab` - 在新标签页打开
   - `Claude Code: Open` - 打开上次的会话

### 方法 3: 侧边栏图标
点击左侧活动栏的橙色 Claude Code 图标

## ⌨️ 常用快捷键

- `Cmd + Escape` - 打开/聚焦 Claude Code
- `Cmd + Shift + Escape` - 在新标签页打开
- `Cmd + N` - 新建对话（在 Claude Code 聚焦时）
- `Alt + K` - 插入 @ 提及引用

## 📝 使用提示

1. **首次使用**: 可能需要登录 Claude 账户
2. **文件引用**: 使用 `@文件名` 来引用项目中的文件
3. **代码选择**: 选中代码后可以直接询问 Claude
4. **设置位置**: 在 Cursor 设置中搜索 "claudeCode" 可以配置插件选项

## 🔧 配置选项

主要配置项（在 Cursor 设置中）：
- `claudeCode.preferredLocation`: 默认打开位置（sidebar/panel）
- `claudeCode.useTerminal`: 是否在终端中使用
- `claudeCode.autosave`: 自动保存文件

---

**提示**: 插件会在 Cursor 启动时自动加载，无需手动启动。
